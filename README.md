# Shinobu

CRA app for planning orders on https://www.mandarake.co.jp/

Powered by **ReactJS**, **Typescript**, **Create React App**, **Web Worker**, **Netlify** & **Netlify Functions**

## Install

```bash
$ yarn install
$ yarn add global netlify-cli
$ netlify init
```

## Development

```bash
$ netlify dev
```
